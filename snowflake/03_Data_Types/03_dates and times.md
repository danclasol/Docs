# Dates and times

### `DATE`

Calendar date (YYYY-MM-DD)

```sql
col_date DATE
```

### `TIME`

Time only (HH:MM:SS)

```sql
col_time TIME
```

### `TIMESTAMP`

Combined date and time (with optional time zone)

```sql
col_ts_ntz TIMESTAMP_NTZ
```

### `TIMESTAMP_NTZ`

No time zone

```sql
col_ts_ntz TIMESTAMP_NTZ
```

### `TIMESTAMP_LTZ`

Local time zone

```sql
col_ts_ltz TIMESTAMP_LTZ
```

### `TIMESTAMP_TZ`

With time zone

```sql
col_ts_tz TIMESTAMP_TZ
```
