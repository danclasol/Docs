# Numbers

### `NUMBER(p, s)`

Fixed-point number (alias: DECIMAL, NUMERIC)

- up to 10 digits
- 2 decimal places

```sql
salary NUMBER(10, 2),
```

### `INT`, `INTEGER`

Integer

- alias of NUMBER

```sql
id INT
```

### `FLOAT`, `FLOAT4`, `FLOAT8`

Floating-point number

- alias: DOUBLE, REAL

```sql
FLOAT
```
