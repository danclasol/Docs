## Common Maven Commands

`mvn clean`

Deletes old build files.

`mvn compile`

Compiles the source code.

`mvn test`

Runs unit tests.

`mvn package`

Packages the app into a JAR/WAR file.

`mvn install`

Installs the package to the local repository.

`mvn spring-boot:run`

Runs a Spring Boot application.
