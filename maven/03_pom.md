## `pom.xml`

Maven uses a configuration file called `pom.xml` (_Project Object Model_) to define:

- Project dependencies (libraries and frameworks)
- Build lifecycle and plugins
- Project metadata (version, name, description, etc.)

# `pom.xml`

## Estructure

- Project Metadata

  - Project name
  - Version
  - File

- Properties

  - Java version

- Dependencies

  - Spring
  - Hibernate
  - Etc

- Plugin
  - Reporting
  - Testing
