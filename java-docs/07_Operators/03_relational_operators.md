# Relational (Comparison) Operators

Compare two values and return a boolean.

### `==`

Equal to

```java
5 == 5 // true
```

### `!=`

Not equal to

```java
5 != 3	// true
```

### `>`

Greater than

```java
5 > 3	// true
```

### `<`

Less than

```java
5 < 3   // false
```

### `>=`

Greater or equal

```java
5 >= 5	// true
```

### `<=`

Less or equal

```java
3 <= 5	// true
```
