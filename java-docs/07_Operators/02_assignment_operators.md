# Assignment Operators

### `=`

Assign

```java
a = 5
```

### `+=`

Add and assign

```java
a += 3 //a = a + 3
```

### `-=`

Subtract and assign

```java
a -= 2 // a = a - 2
```

### `_`=

Multiply and assign

```java
a *= 4 // a = a * 4
```

### `/=`

Divide and assign

```java
a /= 2 // a = a / 2
```

### `%=`

Modulus and assign

```java
a %= 2 // a = a % 2
```
