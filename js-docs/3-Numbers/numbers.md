# Operaciones con numeros

```js
console.log("Sumar", numero1 + numero2);
console.log("Restar", numero1 - numero2);
console.log("Modulo", numero1 % numero2); // Devuelve el resto de la divison
```

# Operadores

Todas estos pares dan el mismo resultado

```js
number1 = numero1 + 2;
number1 += 2;

number1 = numero1 - 2;
number1 -= 2;

number1 = numero1 * 2;
number1 *= 2;

number1 = numero1 / 2;
number1 /= 2;

number1 = numero1 % 2;
number1 %= 2;

number1 = numero1 + 1;
number1++;

number1 = numero1 - 1;
number1--;
```
