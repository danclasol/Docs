# Operadores de asignación

Operadores para guardar información en variables.

## Asignación

Asigna el valor de la parte derecha (en este ejemplo, una suma) a `c`.

```js
const a = 2;
const b = 1;
const c = a + b; // c =3
```

## Suma y asignación

Es equivalente a `a = a + b`

```js
let a = 2;
const b = 1;
a += b; // a=3
```

# Resta y asignación

Es equivalente a `a = a - b`

```js
let a = 2;
const b = 1;
a -= b; // a = 1
```

## Multiplicación y asignación

Es equivalente a `a = a *b`.

```js
let a = 2;
const b = 3;
a *= b; // a = 6
```

# División y asignación

Es equivalente a `a = a / b`.

```js
let a = 6;
const b = 3;
a /= b; // a = 2
```

# Módulo y asignación

Es equivalente a `a = a % b`.

```js
let a = 6;
const b = 3;
a %= b; // a = 0
```

## Exponenciación y asignación

Es equivalente a `a = a **b`.

```js
let a = 2;
const b = 3;
a **= b; // a = 8
```
