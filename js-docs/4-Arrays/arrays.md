# Arrays

Example of array creation

```js
const myArray = [
  "Hola",
  5,
  true,
  undefined,
  { a: 2, b: 3 },
  () => "Hola",
  [1, 2, 3],
];

console.log(myArray);
```

# Access to array elements

```js
console.log(myArray[0]); // Devuelve la primera posicion del array
```

# Modify array elements

```js
myArray[3] = "Pepe";
console.log(myArray);
```
