# Eventos de teclado

## keydown

Cuando pulsamos una tecla

```js
input.addEventListener("keydown", () => {});
```

## keyup

Cuando soltamos una tecla

```js
input.addEventListener("keyup", () => {});
```

## keypress

Cuando pulsamos una tecla y no la soltamos

```js
input.addEventListener("keypress", () => {});
```
