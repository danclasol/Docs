# Schema DDL Commands

## USE schema

Switches to a specific schema for subsequent operations.

```sql
USE my_schema;
```

## CREATE SCHEMA

Creates a new schema.

```sql
CREATE schema my_schema;
```

## DROP SCHEMA

Deletes a schema and all its tables permanently.

```sql
DROP SCHEMA my_schema;
```

## SHOW schemas

```sql
SHOW SCHEMAS;
```
